==========================================
PINWEI SALES TOOLS - SIMPLE WEBSITE
==========================================

HOW TO USE:
-----------
Just double-click "index.html" and it will open in Chrome!

That's it! No npm, no installation needed.

==========================================
FEATURES:
-----------
- Home Screen: Search and browse fabrics
- Matches Screen: View all matching fabrics
- Inquiry Screen: Fill out client preferences
- Favorites Screen: Save fabrics to proposals
- Settings Screen: Admin options

All screens accessible via bottom navigation.

==========================================
FILES:
-----------
- index.html  = Main website (open this!)
- styles.css  = All styling
- script.js   = All functionality
- data.js     = Sample fabric data

==========================================
NOTE:
-----------
You can delete the "node_modules" folder and "src" folder
if you want - they're not needed anymore.
